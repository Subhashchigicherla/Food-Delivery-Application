# Food Delivery Application

A web and mobile-based platform that connects customers with restaurants, allowing users to **browse menus, place orders, and get food delivered seamlessly**. The application provides a robust and user-friendly interface for **customers, delivery agents, and restaurant owners**, demonstrating full-stack development skills.

## Features

- **Customer Interface:** Browse restaurants, view menus, add items to cart, and place orders.
- **Restaurant Management:** Manage menu items, view incoming orders, and update order status.
- **Order Processing:** Cart and checkout functionality with order confirmation.
- **User Management:** Login and signup functionality for customers, restaurants, and delivery agents.
- **Database Integration:** Persistent data storage using MySQL/SQL scripts for users, menus, and orders.
- **Responsive UI:** Web pages built with HTML, CSS, and JSP for dynamic content display.
- **Backend Logic:** Java Servlets handling business logic for orders, users, and menus.
- **Setup Automation:** Includes scripts (`setup_db.bat`, `setup_db2.bat`) for initializing the database.

## Technical Stack

- **Backend:** Java, Servlets, JDBC
- **Frontend:** HTML5, CSS3, JSP
- **Database:** MySQL/SQLite (via provided schema and setup scripts)
- **Tools:** Git/GitHub for version control, PowerShell and batch scripts for setup
